# E-SHOP S NÁBYTKEM

Odkaz na GitHub Pages: https://pslib-cz.github.io/2025-p2b-web-project-Jakub-Macko/

 - Téma: Eshop s nábytkem
 - Obsah: Web bude obsahovat tři základní stánky:
    1. Hlavní stránka (index.html) -- header = jednoduché menu, logo, body = mřížkový seznam všech produktů doplněný fotkou a krátkým popisem, footer
    2. Detail produktu (detail.html) -- rozepsaný popis, parametry produktu, hodnocení a recenze
    3. O nás/kontakty
